# Linux Common Resources

This repository contains cheat sheets and reference guides for common Linux operations, network configuration, and container management with Podman.

## Contents

### 1. [Linux Commands](linux-commands.md)

Comprehensive guide for daily Linux usage, including:

- File & Directory Operations (create, delete, copy, move, search)
- Process Management (kill, monitor, priority)
- System Management (services, system info, monitoring, control)
- Package Management (APT, YUM/DNF)
- Text Processing (grep, sed, awk)

### 2. [Network Reference](network.md)

Detailed reference for network tasks:

- SSH usage (connection, tunneling, config)
- File Transfer (SCP, Rsync)
- Port Management (checking ports, firewalls: UFW, firewalld, iptables)
- Diagnostics (ping, traceroute, DNS, bandwidth)
- Proxy & VPN (SOCKS, WireGuard)

### 3. [Podman Guide](podman.md)

Complete CLI reference for Podman container engine:

- Image & Container Management
- Building Images & Volumes
- Pods & Networks
- System Management & Security
- Practical Examples (Web Server, Database, Node.js)
